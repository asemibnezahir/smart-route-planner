# Smart Route Planner (C++)

A console-based Smart Route Planner built using Object-Oriented Programming and Graph Data Structures.

## Features Completed

- Add City
- Show Cities
- Add Road
- City Validation
- Distance Validation
- Undirected Graph (Adjacency List)

## Upcoming Features

- Duplicate Road Detection
- Show Routes
- DFS (Find All Routes)
- Dijkstra (Shortest Path)
- Algorithm Comparison

## Technologies

- C++
- OOP
- STL
- Vector
- Unordered Map
- Graph (Adjacency List)
